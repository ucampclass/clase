# Instrucciones

 ```
1 Crea un archivo js con el nombre index.js. Este deberas agregarlo a tu archivo html

2 Crea un clase llamada persona. Deberá contener las siguientes propiedades

nombre, edad, sexo

Adicionalmente, agrega un método que se llame saludar. Este método debe imprimir lo siguiente. Hola mi nombre es + la variable nombre 

3 Crea una segunda clase que se llame alumno. Debe extender de la clase persona. Deberas agregas las siguiente propiedades.

matricula, turno, ciclo_escolar

4 Crea una tercera clase que se llame profesor. Debe extender de la clase persona. Deberas agregar las siguiente propiedades.

materia, turno, ciclo_escolar

5 Crea una instancia de alumno. Una vez creada la instancia llena las propiedades nombre, edad, sexo, matricula, turno y ciclo_escolar. Una vez lleno las propiedades, manda llamar el método saludar

6 Al igual que el paso 5, esta vez crea una instancia de la clase profesor. De igual forma llena las propiedades nombre, edad, sexo, materia, turno, ciclo_escolar. Igual, manda llamar al método saludar

*** Genial, ya aprendiste a dominar POO 

```
